# Impdancer 

S2p View and Impedance matching tools

## Windows quick start

Download ```https://github.com/raysan5/raylib/releases/download/5.5/raylib-5.5_win64_mingw-w64.zip``` and place inside src with name raylib-5.5

```console
mingw32-make.exe
build/impedancer path_to_dir_with_s2p_files
```

## Linux quick start
```console
wget https://github.com/raysan5/raylib/releases/download/5.5/raylib-5.5_linux_amd64.tar.gz
mkdir -p src/raylib-5.5
tar -xf raylib-5.5_linux_amd64.tar.gz -C src/raylib-5.5 --strip-components=1
```

To build and run the program (not tested):

```console
mingw32-make.exe
build/impedancer path_to_dir_with_s2p_files
```
